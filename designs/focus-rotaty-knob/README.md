# Focus Rotaty Knob

## Overview

Simple knob made from a hollow cylinder with an attached flange.

## Geometry

- Inner diameter (through-hole): `12 mm`
- Shell thickness: `1 mm` (outer diameter `14 mm`)
- Cylinder height: `5 mm`
- Flange diameter: `20 mm`
- Flange height: `2 mm`
- Total height: `7 mm`

The inner hole penetrates through the full part, including the flange.
