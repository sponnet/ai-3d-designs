# Boxmaker

## Overview

Collection of open-top box and cylinder variants generated from figure dimensions, scaled down for print/export use.

## Geometry

- Produces 7 selectable objects:
  - 5 rectangular open-top box variants
  - 2 cylindrical open-top variants
- Uses a safety wall-thickness strategy before applying final output scale.
