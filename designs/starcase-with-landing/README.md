# Staircase With Landing

## Overview

Parametric U-turn staircase concept with landing, split into two renderable parts for model making and dimension verification.

## Geometry

- Two opposing flights with a landing (U-turn layout)
- Includes treads, risers, and glue tabs
- Uses a deterministic parameter default mode and prints dimension checks during render
