# Foam cutter / utility knife blocks (OpenJSCAD)

- **Source:** `foamcutter.jscad` — top/middle blocks, blade cavity, optional blade visual (`DEBUG`).

From the **repository root**:

```bash
npm run foamcutter:stl   # STL (assembly default)
npm run foamcutter:png   # STL + PNG preview
```

Outputs: `foamcutter.stl`, `foamcutter.png` (and regenerable). See [`../../OPENJSCAD_SKILL.md`](../../OPENJSCAD_SKILL.md).
