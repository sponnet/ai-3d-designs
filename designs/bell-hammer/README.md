# Bell Hammer

## Overview

Replacement hammer for a Brompton bike bell.

## Geometry

- Base body: rounded block (`9 x 18 x 7 mm`)
- Main top pocket: `6.1 mm` diameter, `6 mm` deep, shifted `+2 mm` toward the side hole
- Side bore: `3 mm` diameter, drilled from the small end face to mid-depth
- Opposite-side top relief: `3 mm` step-down on the half away from the side bore
