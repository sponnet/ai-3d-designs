# Staircase (OpenJSCAD)

## Source

- JSCAD source: [`staircase.scad`](./staircase.scad)
- OpenJSCAD: [Open `staircase.scad`](https://openjscad.xyz/?uri=https://raw.githubusercontent.com/sponnet/ai-3d-designs/refs/heads/main/designs/staircase/staircase.scad#)

## Outputs

- Generated artifacts:
  - `staircase.jscad` (temporary copy created by the STL script)
  - `staircase.stl`
  - `staircase.png`

## Commands

From repo root:

```bash
npm run staircase:stl   # STL only
npm run staircase:png   # staircase PNG via unified Blender renderer
npm run renders:png     # render all design STL previews via Blender
```

See [`OPENJSCAD_SKILL.md`](../../OPENJSCAD_SKILL.md) for modeling and export conventions.
